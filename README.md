# STORE2 Theme

CodeJumpの「STORE2」をもとに制作したWordPressオリジナルテーマです。

静的コーディングしたサイトをWordPressテーマとして再構築し、商品一覧ページ、商品詳細ページ、固定ページを実装しました。
